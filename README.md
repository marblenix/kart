# kart

A simple item generator to make games like Jeopardy more interesting.

[Current Master](https://marblenix.github.io/kart)

![Pages Workflow](https://github.com/marblenix/kart/workflows/Release%20workflow/badge.svg)


## Project setup
```
yarn install
```

yarn serve
